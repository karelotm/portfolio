Document Signature & Notification Workflow

This workflow automates the process of sending documents for signature and notifying stakeholders, triggered directly from a Notion database. The system streamlines the final step of client or contract processes by automatically generating and sending DocuSign envelopes when task statuses are updated.

How it Works:

  • • Trigger: Notion database item update
  • • Filter: Validates conditions (email exists, status = "Ready for Signature")
  • • Action: Sends DocuSign envelope using template
  • • Notification: WhatsApp alert to stakeholders

Tools Used:

  • • n8n (Workflow Automation)
  • • Notion (Database & Triggers)
  • • DocuSign (Document Signing)
  • • WhatsApp (Notifications)

Impact: Eliminates manual document processing, reduces errors, and ensures timely contract execution with automatic stakeholder notifications.

RAG AI Chatbot with Automated Knowledge Base

A sophisticated two-part system that builds a complete Retrieval-Augmented Generation (RAG) chatbot. The system can "learn" from documents you provide and answer questions based on that specific knowledge, automatically synchronized from Google Drive.

Workflow A - Knowledge Base Pipeline:

  • • Trigger: Google Drive file changes
  • • Processing: Text extraction and chunking
  • • Vectorizing: Google Gemini embeddings
  • • Storage: Supabase vector database

Workflow B - RAG Chatbot:

  • • Retrieval: Search relevant documents
  • • Generation: Google Gemini responses
  • • Memory: Postgres chat history
  • • Accuracy: Context-based answers only

Impact: Prevents AI hallucination by forcing responses based on specific documents, creating an intelligent assistant that learns from your knowledge base.
